Base de données coach intégrée
Priorités haut niveau : gros exercices d’abord, technique propre, progression simple, fatigue contrôlée, prévention blessures.
Règles foot : pas de grosse séance jambes avant match, cardio réduit si match proche, priorité fraîcheur neuromusculaire.
Règles douleur : réduire charge/amplitude, éviter mouvements douloureux, stop si douleur vive/gonflement/instabilité.
Règles machines : alternatives prévues si machine prise, séances 45/60/75 min, adaptation selon fatigue.
